# D2R
D2R (Detour) - A Web App to get your dream goals to reality

In the Status Quo, we have too much resources and so many plans to learn something new. Yet we end up wasting our time not finishing that one dream skill we wanted to learn.

We are creating a web-app that uses gamification (reward system and micro-progression) and motivational quotes and tips to keep you going and, most importantly, finish that project you are withholding for a long time.
